# Komendy `screen`

1. **Uruchomienie nowej sesji:**
   ```bash
   screen
   ```
2. **Tworzenie nowej sesji z nazwą:**
   ```bash
   screen -S <nazwa>
   ```

3. **Wyświetlenie listy aktywnych sesji:**
   ```bash
   screen -ls
   ```

4. **Dołączenie do sesji:**
   ```bash
   screen -r <nazwa>
   ```

5. **Rozłączenie od sesji:**
   ```bash
   Ctrl + A + D
   ```

6. **Zabicie bieżącego okna:**
   ```bash
   Ctrl + A + K
   ```

7. **Uruchomienie sesji w tle:**
   ```bash
   screen -S <nazwa> -d -m
   ```

8. **Zakończenie sesji:**
   ```bash
   exit
   ```
